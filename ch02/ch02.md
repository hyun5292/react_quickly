# React 첫걸음

```
# 2.1 엘리먼트 중첩
	> createElement()의 첫 번째 매개변수로 두 가지 자료형을 입력할 수 있음
		- 문자열로 작성한 일반적인 HTML 태그(소문자로 작성)
		- React 컴포넌트 클래스 객체(대문자로 작성)
```

```
# 2-2 React 컴포넌트 클래스 생성
	> 컴포넌트(컴포넌트 클래스)
		>> 컴포넌트 클래스를 이용하면 기능을 느슨하게 결합된 부분으로 분리하여 코드를 재사용할 수 있음
		>> 사용자 정의 엘리먼트를 이용하면 이식 가능한 클래스에 논리를 추상화하고 캡슐화할 수 있음
```

```
# 2-3 React 속성(Props) 사용하기
	> 속성
		: 컴포넌트 내부에서는 변경할 수 없는 값
	> 컴포넌트 클래스 코드에서는 'this.props.속성명'으로 접근 가능
```
